<h2>🔌 Getting started</h2>
Clone the project:

```bash
$ git clone https://github.com/Goytai/WebScraping-TISS.git
```

Acess the folder:

```bash
$ cd WebScraping-TISS
```

Follow the steps below:
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn start
```
